# 🚀 Setup PMS UI 

This guide explains how to:

- Install Node.js
- Create a React app using Vite
- Run the development server
- Expose the app to your local network
- Build for production

---

## 📦 1. Install Node Version - v20.12.1

### 🔹 Windows

1. Download Node.js (LTS recommended) from:
   https://nodejs.org
2. Install using the installer.
3. Verify installation:

## 📦Check Node Version
node - v20.12.1
npm - 10.5.0

```bash
node -v
npm -v  
```
## 📦 Installation process
```bash
git clone https://github.com/rajendrakmr/pms-ui.git
cd pms-ui
npm install --legacy-peer-deps
```


## 📦 Run application
```bash
npm run dev 
```
